# 知的高性能メディアシステム研究室

神奈川大学 山本洋太研究室の公式ウェブサイト。

**公開URL:** https://yy-yamamoto-lab.github.io/

## 構成

```
.
├── index.html      トップページ
├── css/style.css   スタイルシート
├── favicon.ico     ファビコン
└── .nojekyll       GitHub Pages の Jekyll 処理を無効化
```

素の HTML / CSS のみで構成。ビルド不要。

## 編集方法

`index.html` と `css/style.css` を直接編集し、`main` ブランチに push すると
数分後に GitHub Pages へ反映される。

## ローカルでの確認

`index.html` をブラウザで直接開く。
